IntelliGUI

About

IntelliGUI is a web front-end which automatically generates and configures user interfaces.

Part internship and part university course, this project was carried out in a unique combination of both a student-driven company and a course for the Computer Science Bachelor at the Radboud University.

I worked in a team of eight, where I was responsible for designing and implementing a modular user interface using Angular.

The front-end solution we created is powered by CGI Arnhem's back-end. The resulting product allows for quick and flexible user interface generation and configuration based on requirements that the user inputs in a simple specification language.
